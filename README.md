# EcoMarket_Javier_Suarez.
EcoMarket es una plataforma web moderna y minimalista enfocada en el consumo consciente y la promoción de un estilo de vida sostenible. El proyecto consiste en un sitio web multipágina que incluye las secciones de inicio, catálogo de productos, información corporativa y un formulario de contacto. Toda la interfaz fue diseñada para ofrecer alternativas ecológicas de primera calidad, destacando por una arquitectura de código limpia, un diseño visual dinámico y una experiencia de usuario fluida que demuestra un uso avanzado de maquetación estructurada sin dependencia de scripts externos pesados.

Para el desarrollo de la plataforma se priorizó el rendimiento nativo del navegador utilizando tecnologías estándar de la industria. Se implementó HTML5 semántico para garantizar una accesibilidad óptima y un excelente posicionamiento en motores de búsqueda, organizando el contenido mediante etiquetas específicas. Los estilos se estructuraron de forma modular en archivos CSS3 independientes, empleando metodologías como Flexbox y CSS Grid para lograr un diseño totalmente responsivo. Además, se destaca el uso de gráficos vectoriales SVG en línea para la optimización del logotipo y un sistema interactivo de filtrado de productos operado al 100% con la lógica de selectores de CSS, eliminando la necesidad de usar JavaScript para tareas básicas de la interfaz.

Para visualizar el proyecto en un entorno local, solo se requiere un navegador web moderno y un editor de código como Visual Studio Code. El método más directo consiste en abrir la carpeta raíz del proyecto, mantener la estructura organizada de los archivos y hacer doble clic sobre el archivo principal para que se ejecute en el navegador. No obstante, para una experiencia de desarrollo profesional, se recomienda utilizar la extensión Live Server desde el editor de código, lo cual levanta un servidor local temporal que recarga la página de manera automática cada vez que se guarda un cambio. Una vez abierto el sitio, se puede navegar de forma fluida entre las diferentes páginas internas utilizando el menú del encabezado.
